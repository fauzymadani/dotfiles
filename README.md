# dotfiles
this is my personal config in debian 12. 
requirements:
`git`, `gnu-stow`.

## usage
first install gnu-stow, on debian based distro:
```bash
sudo apt install stow
```
```bash
git clone https://github.com/fauzymadani/dotfiles.git
```
```bash
cd dotfiles
stow i3 #example
```
